# Aegis RL Cyber Defense Platform

## Overview

Aegis RL is an AI-powered cybersecurity platform built using Machine Learning, FastAPI, React, and NSL-KDD dataset analysis.

The platform provides:

* Attack Detection using Machine Learning
* Dataset Analysis Dashboard
* Threat Intelligence Visualization
* GeoIP Attack Intelligence
* Global Threat Mapping
* Live Network Monitoring
* PDF Security Reports
* Excel Report Export

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts
* React Leaflet

### Backend

* FastAPI
* Scikit-learn
* Pandas
* Joblib

### Machine Learning

* Random Forest Classifier
* NSL-KDD Dataset

## Features

### AI Attack Detection

Predicts network attack types using a trained machine learning model.

### Threat Intelligence

Displays attack distributions and severity levels.

### GeoIP Intelligence

Provides attack source information.

### Global Threat Map

Visualizes attack locations on an interactive map.

### Security Reports

Exports PDF and Excel reports.
